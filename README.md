## 📂 核心功能一览

* 🔍 **智能本地扫描**：一键唤醒隐藏在您手机各个角落的音频文件（支持 MP3, FLAC, WAV 等主流格式）。
* 🗂️ **多维歌曲分类**：支持按歌曲、专辑、歌手、文件夹快速筛选检索，找歌从此不迷路。
* 🌟 **智能播放列表**：内置“最近添加”、“我的收藏”、“听歌排行”等智能化看板，比你更懂你的音乐偏好。
* ➕ **自定义歌单**：支持自由新建本地歌单，随心定制你的专属音乐世界。
